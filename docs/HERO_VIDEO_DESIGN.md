# تصميم الفيديو في الصفحة الرئيسية

## نظرة عامة
تم تحديث تصميم الصفحة الرئيسية لعرض الفيديو بطريقة مبتكرة داخل شكل يشبه رمز الاقتباس (Quote Symbol) مع حركة تلقائية مستمرة.

## المميزات الجديدة

### 1. تخطيط جديد (Grid Layout)
- **النص الإعلاني**: على اليسار (في الشاشات الكبيرة)
- **الفيديو**: على اليمين داخل شكل مميز
- تخطيط متجاوب يتكيف مع جميع أحجام الشاشات

### 2. شكل رمز الاقتباس
```
┌─────────────────────┐
│   "               " │
│                     │
│    [الفيديو]        │
│                     │
│   "               " │
└─────────────────────┘
```

#### العناصر المرئية:
- **علامة اقتباس علوية**: في الزاوية العلوية اليمنى
- **علامة اقتباس سفلية**: في الزاوية السفلية اليسرى (مقلوبة)
- **حركة عائمة**: الرموز تتحرك بشكل بطيء للأعلى والأسفل

### 3. الحركات التلقائية المستمرة

#### أ. الإطار الخارجي
- **دوران بطيء**: الإطار الخارجي يدور ببطء (8 ثواني لدورة كاملة)
- **تأثير التوهج**: لون متدرج من `accent` إلى `accent-dark`
- **تأثير الضبابية**: `blur-xl` لإعطاء تأثير ناعم

#### ب. الحدود المتحركة
- **حدود أفقية**: تتحرك من اليسار لليمين
- **حدود عمودية**: تتحرك من الأعلى للأسفل
- **تأخير متنوع**: لإعطاء تأثير متتابع
- **لون**: `accent` (ذهبي)

#### ج. التأثيرات الضوئية
- **نبضات ضوئية**: حول الإطار بالكامل
- **تدرجات لونية**: من `accent/20` عبر شفاف
- **حركة نبضية بطيئة**: 3 ثواني لكل دورة

#### د. الخلفية المتحركة
- **3 دوائر متحركة**: 
  - دائرة في الأعلى اليسار
  - دائرة في الأسفل اليمين
  - دائرة في المنتصف
- **حركة عضوية**: تتحرك بشكل عشوائي (animation: blob)
- **تأخيرات مختلفة**: 0s, 2s, 4s

### 4. شريط المعلومات
في أسفل الفيديو:
- **نقطة حمراء نابضة**: تشير إلى البث المباشر
- **نص**: "مقاطع من الألعاب التراثية"
- **مؤشرات المقاطع**: نقاط تعرض المقطع الحالي

### 5. النص الإعلاني (على اليسار)

#### العناصر:
1. **Badge**: شارة مع أيقونة نجمة نابضة
2. **العنوان الرئيسي**: 
   - حجم كبير (5xl-7xl)
   - كلمة "الألعاب الشعبية" بتأثير متدرج ذهبي
   - تأثير توهج خفيف
3. **الوصف**: نص توضيحي بحجم كبير
4. **أزرار الإجراءات**:
   - زر "استكشف المعرض" (ذهبي)
   - زر "ابحث عن لعبة" (شفاف مع حدود)

### 6. قسم الإحصائيات
تم نقله إلى قسم منفصل أسفل الفيديو:
- خلفية بيضاء
- بطاقات مع تأثيرات hover
- 3 إحصائيات: الألعاب، الأقاليم، الدول

## الأنيميشن المضافة

### في `globals.css`:

```css
@keyframes spin-slow {
  from { transform: rotate(0deg); }
  to { transform: rotate(360deg); }
}

@keyframes pulse-slow {
  0%, 100% { opacity: 1; }
  50% { opacity: 0.5; }
}

@keyframes shimmer-border {
  0% { transform: translateX(-100%); }
  100% { transform: translateX(100%); }
}

@keyframes shimmer-border-vertical {
  0% { transform: translateY(-100%); }
  100% { transform: translateY(100%); }
}
```

### الكلاسات:
- `.animate-spin-slow`: دوران بطيء (8 ثواني)
- `.animate-pulse-slow`: نبض بطيء (3 ثواني)
- `.animate-shimmer-border`: حركة أفقية للحدود
- `.animate-shimmer-border-vertical`: حركة عمودية للحدود

## التقنيات المستخدمة

### 1. Tailwind CSS
- Grid Layout: `grid lg:grid-cols-2`
- Animations: `animate-*`
- Gradients: `bg-gradient-to-*`
- Backdrop Blur: `backdrop-blur-*`

### 2. SVG
- رموز الاقتباس مرسومة بـ SVG
- مسارات مخصصة (custom paths)

### 3. YouTube IFrame API
- تشغيل تلقائي
- تبديل المقاطع
- بدون أدوات تحكم

## التجاوبية (Responsive)

### الشاشات الصغيرة (< 1024px):
- تخطيط عمودي (column)
- الفيديو أولاً، ثم النص
- حجم خط أصغر

### الشاشات الكبيرة (≥ 1024px):
- تخطيط أفقي (row)
- النص على اليسار، الفيديو على اليمين
- حجم خط كامل

## الأداء

### التحسينات:
- استخدام `transform` و `opacity` للأنيميشن (GPU accelerated)
- `will-change` ضمني في Tailwind
- Lazy loading للفيديو

### الحركات المستمرة:
- جميع الحركات تستخدم `infinite`
- لا توقف تلقائي
- سلسة وغير مزعجة

## الاستخدام

الكود يعمل تلقائياً عند تحميل الصفحة الرئيسية:
- يقرأ مقاطع الفيديو من الإعدادات
- يشغل المقطع الأول تلقائياً
- ينتقل للمقطع التالي عند الانتهاء
- يكرر العملية بشكل لا نهائي

## المتطلبات

- Tailwind CSS 3+
- YouTube IFrame API (يتم تحميله تلقائياً)
