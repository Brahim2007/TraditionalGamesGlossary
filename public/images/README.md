# مجلد الصور

## صورة Hero الرئيسية

### المطلوب:
ضع صورة باسم `hero-poster.jpg` في هذا المجلد لتظهر كخلفية في حال عدم تحميل الفيديو.

### مواصفات الصورة المثالية:
- **الأبعاد**: 1920x1080 (Full HD) أو أعلى
- **الحجم**: يفضل أقل من 500KB للأداء الأمثل
- **الصيغة**: JPG أو WebP
- **المحتوى**: صورة معبرة عن الألعاب الشعبية العربية

### نصائح التصوير:
1. **الموضوع**: أطفال يلعبون ألعاب شعبية
2. **الألوان**: دافئة وجذابة
3. **الإضاءة**: طبيعية وواضحة
4. **التكوين**: تكوين واسع يسمح بإضافة نص فوقه

### بدائل مؤقتة:
يمكنك استخدام صور مجانية من:
- Unsplash
- Pexels
- Pixabay

### ملاحظات:
- الصورة ستظهر أيضاً أثناء تحميل الفيديو
- يتم تطبيق طبقة شفافة داكنة فوق الصورة لتحسين قراءة النص
